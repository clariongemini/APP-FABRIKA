# Monetizasyon — Fabrika Template Modeli

**Fabrika sürümü:** v2.1.0-stable

## Model (fabrika reposu)

Fabrika **MIT lisanslı açık template**; gelir doğrudan repo satışından değil, türetilen uygulamalardan gelir.

| Kanal | Açıklama |
|-------|----------|
| **Template / GitHub** | Ücretsiz — `Use this template`, star/fork büyümesi |
| **Türetilen uygulamalar** | Play Store abonelik / IAP (standart: 7 gün deneme + abonelik) |
| **Kurumsal hizmet** | Özel fabrika sync, governance danışmanlığı (opsiyonel) |

## Uygulama projeleri için standart (şablondan)

`docs/03-STANDARDS/MONETIZATION` ile uyumlu:

- 7 gün ücretsiz deneme
- Abonelik başlangıcında fatura bağlantısı (Play Billing)
- Play Integrity / root detection (güvenlik katmanı)

## Fabrika P0 roadmap bağlantısı

PDC öncelikleri `docs/FACTORY_META/roadmap_priorities.json` — init-governance factory paketi için seed.
