# Pattern Extraction Workflows

## PURPOSE

Outcome, failure, postmortem, venture → **reusable pattern**.

## Workflow

```
07-evidence summary
    → 08-venture results update
    → 06-learning postmortem
    → 08-pattern-extraction analyze
    → experimental/ pattern draft
    → (evidence + 2nd venture) → proven/
```

## Extraction triggers

| Event | Action |
|-------|--------|
| Outcome recorded | Scan for repeatable technique |
| Failure logged | Scan for anti-pattern → NEVER_AGAIN |
| Postmortem complete | Promote or reject pattern candidate |
| Venture archived | Final pattern extraction pass |
